# 🎯 CodeWars Solutions - Dart

![Dart](https://img.shields.io/badge/dart-%230175C2.svg?style=for-the-badge&logo=dart&logoColor=white)
![CodeWars](https://img.shields.io/badge/codewars-%23B1361E.svg?style=for-the-badge&logo=codewars&logoColor=white)

Este repositorio contiene mis soluciones a diversos desafíos de [CodeWars](https://www.codewars.com/), organizados por niveles (Katas) y resueltos utilizando **Dart**.

---

## 🚀 Estructura del Proyecto

El repositorio se divide en carpetas según el nivel de dificultad del desafío:

### 📂 [Kata 7](Kata7/)
*Desafíos de nivel intermedio-bajo para practicar lógica y manipulación de datos.*

- `String ends with.dart`
- `Sum of odd numbers.dart`
- `Sum of two lowest positive integers.dart`
- `Vowel Count.dart`

### 📂 [Kata 8](Kata8/)
*Desafíos fundamentales para dominar la sintaxis básica de Dart.*

- `Area or Perimeter.dart`
- `Array plus array.dart`
- `Beginner - Lost Without a Map.dart`
- `Count of positives_sum of negatives.dart`
- `Remove String Spaces.dart`
- `Return Negative.dart`
- `Reverse List Order.dart`
- `Reversed Strings.dart`
- `Rock Paper Scissors.dart`
- `Sum Arrays.dart`
- `Terminal game move function.dart`

---

## 🛠️ Cómo ejecutar las soluciones

Si tienes instalado [Dart SDK](https://dart.dev/get-dart), puedes ejecutar cualquier solución directamente desde la terminal:

```bash
dart "Kata8/Reversed Strings.dart"
```
